# Shale – Namma Pride

An Android application developed using Kotlin, Jetpack Compose, Firebase, and MVVM Architecture to improve communication and transparency in educational institutions.

## Features

- Student Attendance Monitoring
- Mid-Day Meal Management
- Event Gallery
- Announcements System
- Student Performance Analytics
- Faculty Management
- Multi-language Support
- Firebase Authentication
- Real-time Cloud Database

## Tech Stack

- Kotlin
- Jetpack Compose
- Firebase
- Firestore Database
- Firebase Storage
- MVVM Architecture
- Android Studio

